MODULO FRM - AREA DE BANCOS

FORMULARIO DE CUENTAS BANCARIAS

En este formulario usted podra hacer los mantenimientos necesarios del formualario de Cuentas Bancarias.

En el navegador se le presentan las opciones que usted puede realizar como Insertar, Modificar, Eliminar, Consultas, Imprimir, Ayudas.

El boton de refrescar le permitira recargar el formulario si de algun modo los registros no son actualizados.

El boton de guardar le permitira guardar los cambios despues de presionar el boton de Insertar o Modificar.

El boton de cancelar le permitira cancelar cualquier procesos que este relizando actualmente.

Las flechas le permitiran moverse entre los registros de la tabla del formulario.

El boton de consultas, le permitira abrir la ventana de consultas, en el caso de que usted sea un usuario avanzado se le abrira una ventana distinta.

El boton de ayudas, le permitira abrir la ayuda sobre el formulario.

INGRESAR UN NUEVO REGISTRO

Para ingresar un nuevo registro debera presionar el boton de insertar en el navegador, este le habilitara los campos en el cual usted debera seleccionar lo siguiente: .

1.El banco al cual pertenece la cuenta bancaria.

2.El propietario de dicha cuenta.

3.El tipo de moneda.

4.Ingresar el saldo que la cuenta tiene actualmente.

5.Seleccionar el estado de la cuenta (Activo / Inactivo).

6.Seleccionar la fecha en la que se aperturo dicha cuenta.

Despues de llenar todos los campos necesarios, debera presionar el boton guardar del navegador, para que los datos queden almacenados en la base de datos.

MODIFICAR UN REGISTRO

Para modificar un registro, debera posicionarse sobre el registro que quiere modificar por medio de las flechas del navegador, cuando el registro ya este posicinado en los campos, debera presionar el boton de modificar para que se activen los campos y asi poder modificarlos, cuando termine de realizar la modificacion de los campos, debera presionar el boton guardar para que los datos queden almacenados.

ELIMINACION DE UN REGISTRO

Para eliminar un registro, debera posicionarse sobre el registro que quiere eliminar por medio de las flechas del navegador, cuandel registro ya este posicinado en los campos, debera presionar el boton de eliminar, esto le mostrara un mensaje de advertencia, al aceptar el dato quedara eliminado, de lo contrario no sera eliminado

CANCELAR UN PROCESO

El boton de cancelar, tendra la funcion de cancelar una insercion o modificacion de datos, si el usuario presiona insertar o modificar, pero ya no desea realizar ninguna accion, este boton cancelara dicha accion, y volvera a posicionar el primer registro en los campos.
